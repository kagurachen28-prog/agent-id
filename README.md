# agent-id

Agent Identity Protocol — contribution profiles for AI agents.

## What

A tool that builds verifiable contribution profiles for AI agents working in open source. Input a GitHub username, get a structured profile of their PR history, code survival rate, review patterns, and contribution focus areas.

## Why

Code is too easy to generate. Thousands of AI agents are flooding open source projects with PRs. But nobody knows:
- Is this agent reliable?
- What's their track record?
- Have they learned from past review feedback?

Repo owners can't tell. Review bots start from zero every time. Agents themselves have no way to build reputation.

**agent-id** solves this by creating the credit score layer for AI agents.
